Manual de instalación y uso de
U Stair

U Stair es una aplicación que permite a la comunidad universitaria poder contar con un entorno agradable, que organice sus planes a lo largo de sus programas académicos, organizando las materias que ha visto y querrá ver en los diferentes semestres que curse. ¡Comencemos a explorar todas las herramientas con las que cuenta!

Instalación

Para poder instalar la aplicación, haga uso del instalador, según su sistema operativo.

Windows

Para Windows descargue el instalador .exe y ejecútelo, siga las instrucciones del instalador.

Linux

Distros Basadas en Debian

Para estas distribuciones puede descargar el archivo .deb. Una vez descargado, desde la terminal ubicarse en el directorio donde se encuentra el instalador y correr alguno de los dos siguientes comandos:

$ sudo apt install ./u_Stair.deb

$ dpkg -i u_Stair.deb

Distros Basadas en RPM

El archivo .rpm permite instalar la aplicación en estas distribuciones, por favor descárguelo y ejecute el siguiente comando desde la terminal, en el directorio donde se descargó el instalador:

$ sudo rpm –i u_Stair.rpm

Distros Basadas en Pacman

Para estos sitemas operativos, descargue el instalador .pacman y en la carpeta donde se halla descargado, ejecute el comando:

$ sudo pacman –U u_Stair.pacman

La Aplicación

Una vez la aplicación sea instalada, puede ser ejecutada. Cuando abra la aplicación, aparecerá la ventana de bienvenida, como la que se muestra a continuación.

En esta ventana se podrá crear o abrir mallas, es decir, archivos propios de la aplicación, los cuales tienen extensión .umsh. También se podrán cambiar la configuración de la aplicación. Esto se explicará a mayor detalle en las secciones siguientes.
